1)Download or Clone the Project
The user downloads the project folder from GitHub or runs the git clone command to get all the code files on their computer.
*CCOMMAND:-git clone https://github.com/Jeevesh-Bisht/SignAI-Engine.git
cd SignAI-Engine

2)Place the requirements file in the folder
The user ensures that the requirements.txt file is present in the main root directory of the project folder.

3)Open Terminal or Command Prompt
The user opens their system terminal, command prompt, or terminal inside their code editor and navigates into the project directory.

4)Run the Installation Command
The user types the command pip install -r requirements.txt and presses Enter. Python reads the entire list of libraries from the file and automatically installs all of them with their correct versions at once.
*COMMAND:- pip install -r requirements.txt

5)Run the Application
Once the installation is complete, the user types the command streamlit run app_web.py to launch the real-time project dashboard on their local system.
Run the app_web.py file
